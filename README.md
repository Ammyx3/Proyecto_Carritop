# Visión Robot - Exploración y Detección de Pelota

## Descripción del Proyecto
Robot móvil equipado con cámara Raspberry Pi que explora su entorno, detecta múltiples pelotas y se desplaza hacia una pelota objetivo específica utilizando visión artificial.

## Objetivo
Implementar un sistema de visión por computadora en un robot móvil capaz de explorar, detectar y clasificar objetos en tiempo real, demostrando la integración de hardware y software de visión artificial.
